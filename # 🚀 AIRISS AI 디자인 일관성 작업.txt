🚀 실행 명령어 템플릿

  AIRISS 프로젝트 전환 시작

  # Step 1: Material-UI 설치
  npm install @mui/material @emotion/react @emotion/styled @mui/icons-material

  # Step 2: 기존 UI 프레임워크 제거
  npm uninstall bootstrap tailwindcss [기존_프레임워크]

  # Step 3: 테마 파일 생성
  mkdir src/theme && touch src/theme/index.js

  # Step 4: 컴포넌트 마이그레이션
  npx @mui/codemod v5.0.0/preset-safe src/components
