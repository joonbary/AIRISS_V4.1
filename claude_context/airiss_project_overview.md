# AIRISS 프로젝트 종합 개요

## 🎯 프로젝트 미션
**"AI 기반 과학적 인재관리로 조직의 성과와 공정성을 동시에 실현"**

### 핵심 가치 제안
- 📊 **정량화**: 모든 인재 평가를 객관적 데이터로 수치화
- ⚖️ **공정성**: AI 편향 탐지로 공평한 평가 보장
- 🔮 **예측성**: 미래 성과와 위험 요소 사전 식별
- 🌏 **확장성**: 글로벌 SaaS 플랫폼으로 발전 가능

## 📈 프로젝트 연혁

### Phase 1: 기반 구축 (2024 Q1-Q3)
- ✅ **v1.0**: 기본 텍스트 분석 시스템
- ✅ **v2.0**: 정량 데이터 통합
- ✅ **v3.0**: 8대 차원 평가 체계 확립
- ✅ **v4.0**: 하이브리드 분석 엔진 완성

### Phase 2: 고도화 (2024 Q4-2025 Q2)
- ✅ **v4.1**: 웹 대시보드 및 실시간 분석 (현재)
- 🚧 **v5.0**: AI 고도화 (딥러닝 NLP + 편향탐지 + 예측분석)
- 🎯 **v5.1**: SaaS 플랫폼 전환 준비

### Phase 3: 글로벌 확장 (2025 Q3+)
- 🎯 **v6.0**: 다국어 지원 및 글로벌 표준 준수
- 🎯 **Enterprise**: B2B SaaS 상용화
- 🎯 **Global**: 해외 시장 진출

## 🏗️ 현재 시스템 아키텍처 (v4.1)

### 기술 스택
```
Frontend: HTML5 + JavaScript + Chart.js + Bootstrap
Backend: FastAPI (Python 3.9+)
Database: SQLite (개발용) → PostgreSQL (운영용)
AI/ML: scikit-learn + pandas + 자체 분석 엔진
Deployment: Railway.app (클라우드)
Monitoring: 실시간 WebSocket + 로깅
```

### 핵심 컴포넌트
1. **하이브리드 분석 엔진**
   - 텍스트 분석: 키워드 기반 감정/의도 분석
   - 정량 분석: 통계적 점수 계산
   - 통합 스코어링: 가중평균 기반 종합 점수

2. **8대 평가 차원**
   - 업무성과, 리더십협업, 커뮤니케이션, 전문성학습
   - 태도열정, 혁신창의, 고객지향, 윤리준수

3. **실시간 대시보드**
   - Chart.js 기반 시각화
   - WebSocket 실시간 업데이트
   - 반응형 웹 디자인
