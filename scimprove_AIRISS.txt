🎯 목표: 직원 상세 리포트를 A4 한 페이지로 완전히 정렬된 상태로 PDF 저장 가능하도록 "출력 전용 A4 페이지"를 구현

1. 새로운 라우트 추가
- `/employee/:uid/pdf` 경로에 A4 비율 고정 페이지 구성
- 컴포넌트명: `<EmployeePdfPage />`

2. 디자인 조건
- 전체 크기: 794px × 1123px (A4 @96dpi 기준)
- Tailwind 대신 인라인 CSS 또는 `PDF-safe` 클래스 제한 사용
- 모든 텍스트·박스·레이더차트 위치 고정

3. 자동 캡처 및 다운로드
- URL에 `?download=true` 파라미터 있으면
  - html2canvas → jsPDF → 자동 다운로드 → window.close()

4. 버튼 동작 수정
- 기존 모달 UI 내 [PDF 저장] → 위 경로 새 창 열기 및 자동 다운로드 방식으로 변경

5. 테스트 체크리스트
- 모든 개인의 출력결과가 정확히 1페이지 내 정렬되는지 확인
- 백분위, 점수, 차트, 추천 내용까지 완전 출력
- AI 피드백 한글 깨짐 여부, 자간 오류 없는지 검토
